# Pine Script Strategies

A collection of trading strategies and indicators I've built and backtested in TradingView's Pine Script (v6).

Developed as part of personal research into quantitative strategy and ICT concepts.

## Strategies

| Strategy | Description | Instrument | Timeframe |

| iFVG entry | trades only iFVG's | NQ1! | all timeframes |

| iFVG entry - NY killzone only | trades only iFVG's formed in the NY session | NQ1! | all timeframes |
